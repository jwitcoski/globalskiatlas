Coffee Table Book

A work in progress: Bringing the Global Ski Atlas to print

Project Goals

The Global Ski Atlas coffee table book is a personal project with three main goals:

  1. Learn publishing software such as Adobe InDesign
  2. Showcase examples of my cartographic work
  3. Create offline reference materials for ski holidays and casual browsing

Current Status

This project is still in the early development stages. I'm currently working on:

  • Finalizing the map design templates
  • Selecting resorts to feature
  • Learning InDesign for layout
  • Gathering reference materials

Target Completion: Winter 2025

About the Project

The Global Ski Atlas Coffee Table Book will be a collection of standardized maps and essential information about ski resorts from around the world. Unlike traditional resort maps, which are often beautiful but inconsistent artistic renderings, this atlas will use consistent cartographic techniques to allow for easy comparison between resorts.

The book will feature maps created using QGIS and OpenStreetMap data, providing accurate representations of terrain, lift systems, and trail networks. Each resort will be presented with the same scale and style, making it easy to understand the relative size and layout of different destinations.

Why a Physical Book?

In our digital age, there's something special about having a physical reference that doesn't require Wi-Fi or battery power. This book will be perfect for:

  • Planning ski trips when you're away from internet access
  • Comparing resorts side by side in a tangible format
  • Enjoying beautiful cartography as an art form
  • Sharing with friends and family when discussing ski destinations
  • Having a reference guide during your ski holiday
