Check out the QGIS project to create a map of all 500+ ski resorts in North America: https://github.com/jwitcoski/skiatlas

Before running this model, visualize what the map will look like using OpenSkiMap: https://openskimap.org/

The current model uses polygons for the slopes as well as forest polygons, so ensure they are in OSM before proceeding.

Model: SkiOutputOSM.model3

Plugins needed: QuickOSM (https://plugins.qgis.org/plugins/QuickOSM/), OpenTopography DEM Downloader (https://plugins.qgis.org/plugins/OpenTopography-DEM-Downloader/)

The model will ask for the footage around the ski area you would like to see in the map (default 100 feet) and the extent of the map to download.
